# Product Overview

RuoYi-Vue3-FastAPI is a full-stack rapid development framework based on Vue 3 + FastAPI with front-end and back-end separation.

## Core Features

### Permission Management System
- **User Management**: System user configuration and management
- **Role Management**: Role menu permissions and data scope permissions
- **Menu Management**: System menus, operation permissions, button permissions
- **Department Management**: Organizational structure (companies, departments, groups)
- **Position Management**: User position and job configuration

### System Administration
- **Dictionary Management**: System fixed data maintenance
- **Parameter Management**: System dynamic parameter configuration
- **Notice Management**: System announcement publishing

### Monitoring & Logging
- **Operation Logs**: System operation logging and querying
- **Login Logs**: Login logging including abnormal login attempts
- **Online Users**: Active user status monitoring
- **Server Monitoring**: CPU, memory, disk system information monitoring
- **Cache Monitoring**: Redis cache information and statistics

### Development Tools
- **Scheduled Tasks**: Online task scheduling with execution logs
- **Code Generation**: One-click front-end and back-end code generation from database tables
- **API Documentation**: Automatic API interface documentation generation
- **Form Builder**: Visual form designer
