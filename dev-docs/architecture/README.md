## 核心概念

### 后端架构
- **分层模式**: Controller → Service → DAO → Database
- **异步编程**: 所有数据库操作使用 async/await
- **依赖注入**: FastAPI 依赖系统实现解耦
- **权限控制**: 基于装饰器的授权机制
- **数据权限**: 行级安全的多租户场景

### 前端架构
- **Vue 3 组合式 API**: 使用 `<script setup>` 现代响应式编程
- **TypeScript**: 类型安全开发
- **组件化**: 可复用的模块化 UI 组件
- **Pinia 状态管理**: 集中式状态管理
- **权限指令**: 声明式 UI 访问控制

## 技术栈总结

### 后端
- FastAPI 0.115.8 + Python 3.9+
- SQLAlchemy 2.0.38 (异步 ORM)
- MySQL/PostgreSQL + Redis
- JWT 认证
- APScheduler 任务调度

### 前端
- Vue 3 + TypeScript
- Vite 5.3.2 (构建工具)
- Element Plus 2.7.6 (UI 库)
- Pinia 2.1.7 (状态管理)
- Tailwind CSS 4.1.12

## 开发流程

1. **环境搭建**: 首次运行 `./setup.sh` 初始化环境
2. **启动开发**: 日常开发运行 `./start.sh`
3. **后端开发**: 遵循分层架构模式
4. **前端开发**: 使用组合式 API 和 TypeScript
5. **测试**: 为核心功能编写测试
6. **代码审查**: 遵循最佳实践和编码规范
